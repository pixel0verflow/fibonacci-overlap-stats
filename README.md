# Fibonacci, overlap & stats
Examination task for recruitment
## Description

Write a program which has following functions

(1) fibonacci - takes a non-zero positive integer and returns series up to the highest fibonacci number
that is smaller or equal to the input. Returns "error" if input is wrong or less or equal zero.

(2) overlap - takes list of ranges and prints true if ranges overlap or false if not. Ranges are inclusive.
Returns "error" if any range is incorrect.

(3) stats -  returns "x y" amount of fibonacci and overlap calls
